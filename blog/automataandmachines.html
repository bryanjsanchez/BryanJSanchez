Automata and Machines

Automata and state machines are used to model a systems possible states. The state is determined by the systemâ€™s history. In other words, the current state will determined by the sequence of previous inputs. Each state can read input from the environment and transition into another state, or possibly remain in the same state. These inputs and choices can be deterministic or non deterministic. Automata are useful when you want to represent an entity that has a set of distinct sets in which certain behaviors will affect its state and can be predicted.

It is also possible to represent an automata with infinite states using a pushdown stack automaton or machine. An example of a useful application of infinite state automata is the balancing of parenthesis in a text parser.
